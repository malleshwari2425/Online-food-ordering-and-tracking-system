# 🚀 QuickDine: Full-Stack Food Ordering & Delivery System

QuickDine is a professional, high-performance food delivery platform built with a modern React frontend and a robust Django REST API. It features real-time tracking, multi-role authentication, and high-fidelity food photography.

## ✨ Core Features

### 👤 User Dashboards (Multi-Role)
- **Customers**: Browse restaurants, manage a real-time persistent cart, place orders, and track deliveries.
- **Restaurant Owners**: Dedicated management portal to receive orders and update preparation status.
- **Delivery Partners**: Zone-based job board to accept assignments and mark deliveries as complete.

### 🖼️ High-Fidelity Visual Identity
- **Accurate Menus**: Every dish (from Vada Pav to Tonkotsu Ramen) is mapped to precise, hand-picked Unsplash food photography.
- **Modern UI**: Built with Bootstrap 5.3, Outfit typography, and vibrant gradients.
- **Responsive Design**: Flawless experience across Mobile, Tablet, and Desktop.

### 🚚 Real-Time Logistics
- **Status Timeline**: 4-stage tracking (Placed → Preparing → On the Way → Delivered).
- **Simulated Live Map**: Visual representation of the delivery partner's real-time location.
- **Automated Invoices**: Professional, printable receipts for every order.

### 🇮🇳 Localization
- **Currency**: Fully localized for the Indian market with **₹ (Indian Rupee)** symbols.
- **Realistic Pricing**: Items priced accurately for local market trends (₹60 - ₹750).

## 🛠️ Technical Stack
- **Backend**: Django 5.0, Django REST Framework, SimpleJWT (Auth).
- **Frontend**: React 18, Vite, Axios, Bootstrap 5.
- **Database**: SQLite (Optimized for development).
